# 2.1 Prerequisites

- **.NET 9 SDK** (or later)
- **SQL Server** (Express, LocalDB, or full)
- **Visual Studio 2022** (v17.12+) or **VS Code** with C# Dev Kit
- **Node.js** (optional, for frontend tooling)

# 2.3 Setup Project

Setting up uTPro is simple. Once you have downloaded the source code, you can open the project solution file in Visual Studio.

<img width="260" height="138" alt="image" src="https://github.com/user-attachments/assets/a3075e3f-7ddd-4dba-a891-924ea3cab723" />

Then follow the minimum requirements:

1. **Database**: Create an empty database on SQL Server. Update the connection string in `appsettings.json` under key `umbracoDbDSN`:

<img width="954" height="124" alt="image" src="https://github.com/user-attachments/assets/8b9a772a-5d6c-492b-a3c0-1aaadfdd2585" />

2. **Target Framework**: .NET 9 (matching Umbraco 16.x)

<img width="847" height="307" alt="image" src="https://github.com/user-attachments/assets/ffd40c2a-a951-4910-8350-3c91ee1450db" />

3. **Build & Run**:
```bash
dotnet build uTPro/uTPro.sln
dotnet run --project uTPro/Project/uTPro.Project.Web/uTPro.Project.Web.csproj
```

4. Navigate to `https://localhost:44381/umbraco` to complete Umbraco installation.
